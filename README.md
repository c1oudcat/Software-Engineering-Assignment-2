# Software-Engineering-Assignment-2
軟體工程導論期末作業

=======================================我是分隔線========================================

DataBase.py：
修改DataFile的程式，執行後有根據輸入的數字(1~5)分別有五個功能
1.新增資料，依序填入日期(YYYY-MM-DD)、類別、值(float)，輸入完畢程式會為這組資料生成uuid
2.列出資料，會列出所有資料各自的uuid、日期、類別、值為多少
3.更新資料，輸入資料的uuid，更新資料的日期、類別、值並覆寫回DataFile
4.刪除資料，輸入資料的uuid，刪除其所代表的那組資料
5.退出執行，即退出

=======================================我是分隔線========================================

illustrator.py
僅有一個功能，將DataFile的資料根據值(value)的占比畫成圓餅圖，會在所佔的區塊旁附註資料的類別

merge.py
